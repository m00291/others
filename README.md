
Set Notepad++ as Your Git Editor  
CMD:  
```sieve
git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -nosession"
```  

Verify the Configuration  
To check if the editor has been set correctly, use:  
```sieve
git config --global --get core.editor
```

```sieve
git config --global user.name "User"  
git config --global user.email "user@example.com"
```

1. Replace n with the number of commits you wish to modify:
   ```sieve
   git rebase -i HEAD~n
   ```
   or change from First Commit:
   ```sieve
   git rebase -i --root
   ```
2. Change pick to edit:
   ```sieve
   pick 348f312 Add files via upload
   pick 7d010e5 Create README.md
   ```
   ```sieve
   edit 348f312 Add files via upload
   edit 7d010e5 Create README.md
   ```
3. Then, for each commit:
   ```sieve
   git commit --amend --author="User <user@example.com>"
   ```
   or
   ```sieve
   git commit --amend --author=" "
   ```
5. Continue the rebase:
   ```sieve
   git rebase --continue
   ```
6. Force Push to rewrite history:
   ```sieve
   git push origin --force
   ```
